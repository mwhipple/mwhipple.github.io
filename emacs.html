  • tool-bar-mode The Emacs ToolBar is a bit of chrome I tend to turn off (to be cleaned up a bit later). I may revisit this if I venture into using Emacs more without a keyboard. Internally disabling is done in 
    tool-bar-mode.el
    by setting the lines allocated for the toolbar in the frame parameters to be 0. This wiki page provides some additional notes without anything too insightful.
